Configuration
.
The configuration file is used to configure your installation of EZPHP. It can be found at config/config.php. You can configure the settings for:

  • application
    • default_controller → used to specify the default controller when site is visited.
    • timezone → used to specify the time zone.
  • security
    • error_reporting → used to set error reporting level.
    • display_errors → whether or not errors should be displayed.
    • xss_filter → used to set on or off the XSS (Cross-site scripting) filtering.
  • database
    • use_db → used to specify whether or not the site will need a database.
    • db_class → The database/mysql wrapper class to be used present in classes folder.
    • and more settings related to database connection.
  • template
    • use_template → whether or not site will use a template.
    • template_dir → the name of the template folder present inside the template folder.
  • cache
    • enable_cache → whether cache should be on or off.
    • cache_lifetime → time a page will be cached for.
  • url
    • url_suffix → used to specify a suffix for the site urls.

You should specify 1 for enabling a certain option and 0 for disabling it.
