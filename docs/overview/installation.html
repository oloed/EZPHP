Installation
.
Here are the steps required to install the EZPHP:

  1. Download EZPHP.
  2. Unzip the EZPHP folder and upload to your server.
  3. Configure your installation.

If everything went successfully and you go to site's URL, you should see Welcome page.
.
Having Problems?
.
If you are witnessing any problems, make sure that EZPHP directories are set with correct permissions. Chmod them all to 755.
